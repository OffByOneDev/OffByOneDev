# About Me

I picked up programming as a hobby in 2011 and have since built up a broad foundation across languages, architectures and systems.

When writing code I want to know, in detail, what exactly happens internally. This leads me to favor lightweight and minimal solutions over heavy libraries and frameworks, which consequently results in less deprecations and version mismatches, more readable code, independence and increased long-term maintainability at the cost of a larger codebase. Despite my slight aversion to 3rd party code, I'm still comfortable working with established frameworks when a project calls for them.

My native language is German and I am fluent in English.
